## GB Fantasy App - Live Scores System

### What Was Built
- Simplified FPL API integration for live match scores
- Frontend polling every 2 minutes when user is logged in
- GitHub Actions cron as backup every 30 minutes during match hours
- Time-based finish detection (110 minutes after kickoff)
- Manual refresh button on profile page
- Points calculation when matches finish

### Key Technical Decisions
- Free tier only: GitHub Actions + Vercel + Supabase
- No OpenClaw cron jobs (would cost money)
- Frontend polling keeps data fresh for all users when any user is online
- Time-based fallback for when FPL API is slow to mark matches finished
